# TimeMate
Windows向け多機能時計アプリです。Win11対応。エレクトロォンとの～どジェイエスの練習用につくったので、よほどのことがない限りアプデとかはしないつもりです。
# 実行ファイル名
timemate.exeです。周りの雑多なファイルを消すと動かなくなります。
